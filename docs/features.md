# Tech Hub 機能一覧

## プロジェクト概要

Tech Hub は、フロントエンド学習プラットフォーム兼技術ブログシステム。記事の執筆・公開、インタラクティブなカリキュラム学習、スキルシート管理を提供する。

- フレームワーク: Next.js 15+ (App Router)
- 言語: TypeScript
- DB: PostgreSQL (Supabase)
- ORM: Prisma
- 認証: Supabase Auth
- スタイリング: Tailwind CSS v4 + shadcn/ui
- 3D: Three.js + @react-three/fiber
- AI: OpenRouter (Gemini Flash)
- テスト: Vitest + Playwright
- アーキテクチャ: DDD (ドメイン駆動設計)

---

## 1. 記事公開システム (Publishing Context)

### 記事管理

- 記事の作成 (下書き状態で作成)
- 記事の編集 (タイトル・本文・スラッグ・タグ)
- 記事の公開・非公開切り替え
- 記事の削除 (確認ダイアログ付き)
- マークダウンエディタ (CodeMirror ベース、リアルタイムプレビュー)
- 画像のドラッグ&ドロップ挿入 (Supabase Storage)
- スラッグの自動生成・一意性検証

### エンゲージメント

- 閲覧数カウント (ページ表示時に自動記録)
- いいねトグル (ユーザーごとに1回)
- ブックマークトグル
- ブックマーク一覧ページ

### 記事一覧

- カーソルベースページネーション (デフォルト20件、最大50件)
- ソート (公開日時・閲覧数・いいね数)
- タグフィルタリング
- レスポンシブグリッド (1〜3カラム)

### 管理画面

- 全記事一覧 (ステータス・閲覧数・いいね数表示)
- 公開/非公開/削除操作
- メディアライブラリ (画像アップロード・フォルダ管理)

## 2. カリキュラム学習システム

### カリキュラム一覧

- Three.js 3D背景 (BotW風の風景)
- GLBモデル読み込み対応 (Meshy.ai で生成した城モデル)
- ガラスモーフィズムのカリキュラムカード
- 進捗バー表示

### ステージマップ

- Three.js 3D城塔 (RPG風)
- ステージごとの色分け (完了=金、進行中=青、解放済み=緑、ロック=灰)
- OrbitControls (ドラッグ回転・ズーム)
- 自動回転
- ステージクリックで学習ページへ遷移

### インタラクティブ学習 (Learn Workspace)

- コードエディタ (HTML/CSS タブ切り替え)
- リアルタイムプレビュー (iframe)
- ステップバイステップガイド
- ヒントモーダル (前へ/次へナビゲーション、ドットインジケーター)
- 採点モーダル (採点中アニメーション → 合否表示)
- 失敗時のフィードバック (failureHint による具体的な指摘)
- レッスン画面 (解説のみのステップ)
- プログレスバー

### 採点システム

- getComputedStyle ベースの CSS プロパティ検証
- CheckRule 型: equals / not-equals / includes
- failureHint による具体的なエラーメッセージ
- GradeResult: { passed, failedHints }

### 現在のコンテンツ

- 背景画像の表示をマスターしよう (background-image チャプター)
  - 導入解説 3ステップ
  - 実践 4ステップ + チェックポイント

### 参考記事

- 歯車FABから開くドロワー
- マークダウンレンダリング
- よく使うGitコマンド / プルリクの作成方法 / VSCodeショートカット

## 3. スキルシート管理

- スキルシート作成・編集
- プロジェクト経歴管理 (期間・説明・担当工程)
- スキル分類 (言語・FW・インフラ・ツール・OS)
- 環境情報管理
- PDF出力 (印刷ページ)
- Excel出力 (XLS形式)
- コピー機能 (既存シートの複製)
- 令和日付フォーマット対応

---

## 4. AI アシスト

- 校正 (誤字脱字・不自然な表現の修正)
- 要約 (3文に凝縮)
- 翻訳 (英日・日英)
- リライト (読みやすさ改善)
- タイトル生成 (3候補)
- タグ提案 (5候補)
- レート制限 (10リクエスト/分/ユーザー)
- OpenRouter 経由 Gemini Flash

---

## 5. ユーザー管理

- Supabase Auth (メール/パスワード認証)
- プロフィール管理 (表示名・アバター)
- DiceBear アバタークリエイター
- マイページ (プロフィール・ブックマーク・スキルシート・カリキュラムへのリンク)

---

## 6. 共通機能

### レイアウト

- 公開レイアウト (ヘッダー + フッター)
- 管理レイアウト (ヘッダー + サイドバー)
- モバイルナビゲーション (ハンバーガーメニュー)

### 技術基盤

- マルチテナント設計 (全エンティティに tenantId)
- ULID ベースの ID 採番
- DDD レイヤー構成 (domain / application / infrastructure / presentation)
- Server Actions によるバックエンド通信
- Zod バリデーション (プレゼンテーション層のみ)
- ドメインイベント発行

### アクセシビリティ

- キーボード操作対応
- aria-label 属性
- フォーカス管理
- 適切な見出し階層
