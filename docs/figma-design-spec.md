## 実装に必要なHTML/CSSスキル・知識

### 基礎スキル

| スキル                     | 用途                                                        |
| -------------------------- | ----------------------------------------------------------- |
| Flexbox                    | Header・ButtonBar・フォームラベル行・商品カード内レイアウト |
| CSS Grid                   | 商品一覧（PC: 4列グリッド）                                 |
| position: fixed / absolute | Header固定・モーダルオーバーレイ                            |
| z-index管理                | モーダル表示時の重なり順                                    |
| box-sizing: border-box     | 全要素の基本設定                                            |

### レスポンシブ対応

| スキル                 | 用途                              |
| ---------------------- | --------------------------------- |
| メディアクエリ         | PC(1440px) / SP(390px) の切り替え |
| モバイルファースト設計 | SP基準で記述しPCを上書き          |
| ハンバーガーメニュー   | SP Header → サイドメニュー開閉    |
| グリッド列数変更       | PC: 4列 → SP: 1列                 |

### フォーム実装

| スキル                                   | 用途                                         |
| ---------------------------------------- | -------------------------------------------- |
| input / textarea / select のスタイリング | お問い合せフォーム・購入手続き               |
| appearance: none                         | セレクトボックスのデフォルトスタイルリセット |
| カスタムラジオボタン                     | 支払い方法選択（楕円形、選択色変更）         |
| カスタムチェックボックス                 | プライバシー同意                             |
| バリデーションエラー表示                 | 赤テキスト（12px）の条件付き表示             |
| placeholder スタイリング                 | ::placeholder 疑似要素                       |

### モーダル実装

| スキル                        | 用途                           |
| ----------------------------- | ------------------------------ |
| position: fixed + inset: 0    | オーバーレイ全画面表示         |
| overflow: hidden（body）      | モーダル表示中のスクロール防止 |
| display: none / flex 切り替え | モーダルの開閉                 |
| backdrop（半透明黒）          | rgba(0,0,0,1) オーバーレイ     |

### テーブル実装

| スキル                          | 用途                                  |
| ------------------------------- | ------------------------------------- |
| table / thead / tbody / tr / td | カート一覧テーブル                    |
| border-collapse                 | テーブルボーダー結合                  |
